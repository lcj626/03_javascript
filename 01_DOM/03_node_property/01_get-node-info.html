01. 노드 정보 취득

Node.prototype.nodeType : 노드 객체의 종류, 즉 노드 타입을 나타내는 상수 반환 Node.Element_node : 요소 노드 타입을 나타내는 상수 1반환 Node.Text_Node : 텍스트 노드 타입을 나타내는 상수 3반환 Node.Document_node : 문서 노드 타입을 나타내는 상수 9반환 Node.prototype.nodeName: 노드의 이름을 문자열로 반환한다. 요소 노드 : 다문자 문자열로 태그 이름을 반환 텍스트 노드 : 문자열 #text를 반환 문서 노드 : 문자열 #document를 반환
