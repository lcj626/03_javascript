02. className과 classList

        class 어트리뷰트에 대응하는 Dom 프로퍼티는 class가 아니라 className, classList이다.
        요소 노드의 className 프로퍼티는 setter와 getter가 모두 존재하는 접근자로 프로퍼티로 요소 노드의
        className 프로퍼티를 참조하면 class 어트리뷰트의 값을 문자열로 반환하고, 
        요소 노드의 className프로퍼티에 문자열을 할당하면 class 어트리뷰트 값을 할당한 문자열로 변경한다.
        Element.prototype.classList 프로퍼티는 class 어트리뷰트의 정보를 담은 DomTakenList 객체를 반환한다.
